# CodeAlpha - Stock Portfolio Tracker

## Project Description

The Stock Portfolio Tracker is a simple Python-based application that helps users calculate the total value of their stock investments. The program uses a predefined dictionary to store stock prices and allows users to enter a stock name and quantity. Based on the user input, the application calculates and displays the total investment value.

## Objective

The objective of this project is to understand and implement Python concepts such as dictionaries, input/output operations, conditional statements, and arithmetic calculations in a real-world financial application.

## Features

* Supports AAPL and TSLA stocks.
* User-friendly interface.
* User can enter any quantity of shares.
* Calculates total investment value automatically.
* Uses predefined stock prices for quick calculations.

## How to Run the Program

1. Install Python 3.x on your system.

2. Open the project folder in Visual Studio Code.

3. Open the terminal.

4. Run the following command:

   python stock_tracker.py

5. Enter the stock name (AAPL or TSLA).

6. Enter the quantity of shares.

7. View the calculated investment value.
